# 채팅 시스템

로비와 방(대기실/게임룸)에서 사용되는 실시간 채팅.

## 채팅 종류

| 종류 | 위치 |
|------|------|
| 로비 채팅 | 로비 화면 |
| 방 채팅 | 대기실 + 게임룸 |

## 메시지 제한

| 항목 | 값 |
|------|-----|
| 메시지 최대 길이 | 500자 |
| 서버 이력 저장 | 로비/방 각 최근 50개 |
| 클라이언트 메모리 | 최근 100개 |

## 이력 요청

로비/방 입장 시 서버에 이력을 요청하여 최근 50개 메시지를 불러온다.

## 시스템 메시지

시스템이 자동 생성하는 메시지. 채팅 패널에서 중앙 정렬 메인 컬러(primary) 배지로 표시된다. 서버에서 생성되어 이력에 저장되므로 재접속 시에도 확인 가능하다.

### 입퇴장 알림

서버에서 생성하여 방 채팅 이력에 저장한다. 대기실과 게임룸 모두 동일하게 동작한다.

| 상황 | 메시지 |
|------|--------|
| 플레이어 입장 | `OOO님이 입장했습니다.` |
| 플레이어 퇴장 | `OOO님이 퇴장했습니다.` |
| 관전자 입장 | `OOO님이 관전을 시작했습니다.` |
| 관전자 퇴장 | `OOO님이 관전을 종료했습니다.` |

### 캐치마인드 정답 알림

서버에서 생성하여 방 채팅 이력에 저장한다.

- 형식: `OOO님이 정답을 맞추었습니다! (1등, +3점)`
- 정답 메시지 자체는 채팅에 표시하지 않는다 (정답 노출 방지)

## 관전자 채팅

- 호스트가 "관전자 채팅" ON/OFF 설정 (기본값 OFF)
- OFF이면 서버에서 관전자 메시지를 차단하고, 입력창을 비활성화한다
- ON이면 관전자 메시지에 `[관전]` 배지를 표시한다

## 관리자 메시지

- 관리자 닉네임으로 인증된 사용자의 메시지
- 닉네임이 "관리자"로 변환되어 표시된다
- 닉네임 색상: 빨간색 (일반 사용자는 하늘색)

## 관리자 메시지 삭제

- 관리자만 로비 채팅 메시지를 삭제할 수 있다
- 방 채팅 메시지는 삭제 불가

## 캐치마인드 채팅 규칙

그리기 페이즈에서 적용되는 특수 규칙:

| 대상 | 규칙 |
|------|------|
| 출제자 | 채팅 입력 비활성화 (정답 힌트 방지) |
| 정답 맞춘 플레이어 | 채팅 입력 비활성화 (정답 노출 방지) |
| 나머지 플레이어 | 채팅 입력이 정답 체크 대상이 됨 |

- 그리기 페이즈가 아닌 경우(라운드 결과 등)에는 모든 플레이어가 자유롭게 채팅 가능

## UI 표시

| 요소 | 스타일 |
|------|--------|
| 내 메시지 | 우측 정렬 |
| 상대 메시지 | 좌측 정렬 |
| 시스템 메시지 | 중앙 정렬, 메인 컬러(primary) 배지 |
| 관전자 메시지 | `[관전]` 배지 (주황색) |
| 관리자 닉네임 | 빨간색 |
| 일반 닉네임 | 하늘색 |
| 타임스탬프 | YYYY-MM-DD HH:mm:ss |

## 귓속말

특정 사용자에게 1:1 비밀 메시지를 보내는 기능. 다른 사용자에게는 노출되지 않는다.

### 사용 방법

1. 채팅 입력창에 `@`를 입력하면 접속자 닉네임 자동완성 드롭다운이 표시된다
2. 드롭다운에서 대상을 선택하면 `@닉네임 ` 형태로 자동 입력된다
3. 이어서 메시지를 작성하고 전송한다

### 자동완성 드롭다운

- `@` 입력 시 현재 접속 중인 전체 사용자 목록을 표시한다
- 자기 자신은 목록에서 제외된다
- `@` 뒤에 텍스트를 입력하면 닉네임 필터링이 적용된다
- 키보드(위/아래 화살표, Tab/Enter)와 마우스 클릭으로 선택 가능하다
- Escape로 드롭다운을 닫을 수 있다

### 전송 결과

| 대상 | 동작 |
|------|------|
| 보낸 사람 | 토스트 알림으로 전송 완료 표시 |
| 받는 사람 | 토스트 알림으로 메시지 내용 표시 |
| 채팅창 | 아무것도 표시되지 않음 |

### 제한사항

- 메시지 최대 길이: 500자
- 이력을 저장하지 않는다 (휘발성)
- 로비/방 구분 없이 접속 중인 모든 사용자에게 전송 가능하다
- 접속 중이 아닌 사용자에게는 전송할 수 없다
- 자기 자신에게는 전송할 수 없다
- 관리자가 보낸 귓속말은 닉네임이 "관리자"로 표시된다

## 한글 IME 처리

한글 입력 시 합성(composition) 중 엔터 키를 무시하여 마지막 글자 중복 입력을 방지한다.
