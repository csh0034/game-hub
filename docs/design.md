# 디자인

Neon Arcade 컨셉의 다크 게이밍 UI 테마를 사용한다.

## 컨셉

네온 아케이드 라운지를 모티브로 한 디자인이다. 깊은 다크 배경 위에 네온 컬러가 발광하는 느낌을 준다. 배경에는 미세한 그리드 패턴이 있어 아케이드 분위기를 연출한다.

## 컬러

| 역할 | 색상 | 용도 |
|------|------|------|
| 배경 | 딥 네이비 | 전체 배경 |
| 카드 | 다크 네이비 | 카드, 패널 배경 |
| 주색 (시안) | 일렉트릭 시안 | 버튼, 활성 탭, 링크, 글로우 효과 |
| 보조색 (핑크) | 핫 핑크 | 경고, 삭제, 관리자 표시 |
| 보조색 (퍼플) | 네온 퍼플 | 공지사항, 오픈 예정, 그래디언트 |
| 성공 | 네온 그린 | 연결 상태, 성공 표시 |
| 경고 | 네온 옐로 | 주의, 점검 중, 트로피 |

- 주색(시안)과 보조색(퍼플)을 조합한 그래디언트를 로고와 주요 아이콘에 사용한다
- 네온 컬러는 글로우(발광) 효과와 함께 사용하여 네온사인 느낌을 낸다

## 타이포그래피

| 용도 | 서체 | 특징 |
|------|------|------|
| 헤딩/로고 | Chakra Petch | 테크/게이밍 느낌의 디스플레이 서체 |
| 본문/UI | Noto Sans KR | 한글 가독성이 높은 본문 서체 |
| 코드/수치 | JetBrains Mono | 고정폭 서체, 랭킹 점수 등에 사용 |

- 헤딩에는 대문자 영문과 넓은 자간을 사용한다 (GAME HUB, CHAT, RANKING 등)
- 탭, 버튼 등 UI 요소에 Chakra Petch 서체를 적용한다

## 네온 효과

- 카드와 패널은 호버 시 시안 보더 글로우가 나타난다
- 주요 버튼은 호버 시 시안 섀도 글로우가 강해진다
- 로고 아이콘은 시안-퍼플 그래디언트 배경에 상시 글로우가 적용된다
- 닉네임 입력 화면의 로고 아이콘은 펄스 애니메이션으로 글로우가 반복된다
- 네비게이션 바 전체에 은은한 시안 글로우가 적용된다

## 다이얼로그

| 종류 | 테마 색상 | 용도 |
|------|-----------|------|
| 확인 | 옐로 | 게임 나가기 등 경고성 확인 |
| 공지 전송 | 시안 | 관리자 공지 작성 |
| 공지 수신 | 퍼플 | 공지사항 오버레이 |
| 플랜카드 | 시안 | 배너 문구 설정 |
| 도움말 | 시안 | 게임 규칙 안내 |

- 각 다이얼로그는 테마 색상에 맞는 보더 글로우를 가진다
- 확인(삭제/나가기) 버튼은 핑크 색상을 사용한다

## 배경

- 전체 배경에 시안 색상의 미세한 그리드 라인 패턴이 깔린다
- 닉네임 입력 화면에는 시안과 퍼플의 블러 처리된 원형 장식이 배치된다

## 파비콘

시안-퍼플 그래디언트 배경 위에 게임패드 아이콘을 표현한 SVG 파비콘을 사용한다.

## OG 이미지

- 다크 네이비 배경에 시안 그리드 패턴
- 시안-퍼플 그래디언트의 "GAME HUB" 타이틀
- 게임패드 아이콘과 6개 게임 아이콘 카드
- 하단에 시안-퍼플 그래디언트 악센트 라인
