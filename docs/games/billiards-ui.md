# 3쿠션 당구 UI 및 3D 렌더링

게임 허브에 구현된 3쿠션 당구의 3D 렌더링 및 UI 명세이다.

## 3D 라이브러리

| 라이브러리 | 용도 |
|------|------|
| three | 3D 렌더링 코어 |
| @react-three/fiber | Three.js의 React 바인딩 |
| @react-three/drei | OrbitControls, PerspectiveCamera 등 유틸 |

- 물리는 순수 TypeScript 엔진을 사용한다 (ammo.js 미사용)
- 모든 3D 형태는 Three.js 프리미티브로 코드에서 직접 생성한다 (외부 텍스처/모델 파일 없음)

## 당구대

| 부품 | 색상 |
|------|------|
| 천(cloth) | 파란색 (#1d4ed8) |
| 프레임(다이) | 갈색 (#3a2a1f) |
| 쿠션 | 파란색 (#2d57dc) |
| 다이아몬드 마커 | 아이보리 (#d7d0c2), 장축 9개 단축 5개 |

- 쿠션 변형: 공이 쿠션에 충돌 시 스프링-댐퍼 애니메이션으로 찌그러짐

## 큐(당구채)

| 파트 | 색상 |
|------|------|
| 팁 | 파란색 (#5ba3d9) |
| 페룰 | 아이보리 (#f0ece0) |
| 샤프트 | 나무색 (#e8c882) |
| 그립 | 적갈색 (#6b3a2a) |

- 큐 팁이 수구 표면(당점)에 정확히 닿도록 위치 계산
- 대기 상태에서 큐 팁과 수구 표면 사이 기본 간격: 6cm
- 파워 드래그 중 최대 0.3m 후퇴
- 샷 시 200ms 동안 앞으로 치고 복귀하는 애니메이션

## 카메라 시스템

### 추적 카메라 (내 턴 조준 중)

- 큐볼 뒤쪽에서 공을 바라보는 3인칭 카메라
- 기본 고도 약 31°, 거리 2.5m
- 우클릭 좌우 드래그: 조준 방향 + 카메라 동시 회전
- 우클릭 상하 드래그: 카메라 고도 조절 (위로 드래그 시 시점이 낮아짐)
- 휠로 줌인/아웃 (거리 1.2~6m)
- 새 턴 시작 시 가장 가까운 목적구 방향으로 자동 조준 (카메라 고도/거리는 유지)
- 샷 직후 시뮬레이션 시작까지 500ms 동안 카메라 위치 고정

### 자유 카메라 (시뮬레이션 중 / 다른 사람 턴 / 관전)

- OrbitControls 활성화 (우클릭 드래그 회전, 휠 줌)
- 거리 1.2~6m, 수직 각도 0~72도 제한, 패닝 비활성화
- 활성화 시 이전 카메라 위치를 그대로 이어받아 점프 없음

## HUD 오버레이

Canvas 위에 position: fixed 레이어로 표시된다.

| 요소 | 위치 | 내용 |
|------|------|------|
| 점수판 | 좌상단 | 플레이어별 점수, 현재 턴 하이라이트, 남은 시간 |
| 쿠션 카운터 | 상단 중앙 | 시뮬레이션 중 쿠션 1/2/3 원형 인디케이터 + 목적구 히트 수 |
| 옵션 패널 | 우상단 | 공 궤적 토글 스위치 (항상 노출) |
| 당점 표시 | 좌하단 | 공 단면도에 빨간 점 + 오프셋 표시, 미스큐 경고 |
| 파워 게이지 | 하단 중앙 | 드래그 중에만 표시, 가로 바 + 색상 그라데이션 + %/km/h |
| 가이드라인 | 3D 씬 | 미니 물리 시뮬레이션 기반 경로 예측 (점선) |
| 조작 안내 | 하단 중앙 | 내 턴: 조작법, 상대 턴: 차례 표시 |
| 샷 결과 | 화면 중앙 | 득점/MISS + 쿠션 횟수 팝업 |
| 채팅 패널 | 우하단 | 드래그 가능 팝업, 접기/펼치기 가능 |

## 기타

- 3D 씬은 클라이언트에서만 렌더링한다 (SSR 제외)
- 터치 입력 미지원 (마우스/키보드만 지원)
