# 3쿠션 당구 (Billiards)

3쿠션 당구 게임. 수구로 두 목적구를 모두 맞히되, 쿠션을 3회 이상 터치해야 득점한다. 풀스크린 3D로 렌더링된다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 1~8명 |
| 목표 점수 | 10점 (방 생성 시 설정 가능) |
| 턴 제한 시간 | 30초 (방 생성 시 설정 가능, 10~60초) |
| 공 개수 | 3개 (수구 1, 목적구 2) |

## 게임 진행

1. 방장이 게임을 시작하면 3개의 공이 초기 위치에 배치된다
2. 플레이어 1(방장)이 먼저 샷을 한다
3. 조준 방향과 힘을 설정하여 수구를 친다
4. 수구가 두 목적구를 모두 맞히고, 두 번째 목적구 접촉 전까지 쿠션 3회 이상이면 득점
5. 득점 시 같은 플레이어가 연속으로 친다
6. 미득점 시 상대에게 턴이 넘어간다
7. 목표 점수에 먼저 도달한 플레이어가 승리한다

## 화면 구성

당구는 풀스크린 3D 레이아웃을 사용한다. 상단 내비게이션 바와 하단 사이의 전체 영역이 3D 캔버스로 채워진다.

- 점수판, 턴 표시, 쿠션 카운터 등은 3D 화면 위에 오버레이로 표시된다
- 채팅은 드래그 가능한 팝업 패널로 제공되며, 접기/펼치기가 가능하다

## 게임 중 이탈

한 명이 나가면 남은 플레이어가 자동 승리한다.
