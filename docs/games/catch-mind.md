# 캐치마인드 (Catch Mind)

## 개요

3~8인 드로잉 퀴즈 게임. 출제자가 제시어를 그림으로 표현하면, 나머지 플레이어들이 **채팅으로 정답을 맞춘다**. 정답을 맞춘 플레이어가 점수를 획득한다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 3~8명 |
| 출제자 수 | 1명 (라운드마다 교체) |
| 그리기 시간 | 1~120초 (방장 설정, 기본 60초) |
| 라운드 수 | 1~10회 (방장 설정, 기본 3라운드) |
| 글자 수 힌트 | ON/OFF (방장 설정, 기본 OFF) |

## 게임 흐름

### 1. 라운드 시작 — 출제자 선정

1. 플레이어 중 **1명이 출제자**로 선정된다.
2. 출제자에게 **제시어**가 공개된다.
3. 나머지 플레이어에게는 제시어가 숨겨진다.
4. 글자 수 힌트 옵션이 ON이면, 나머지 플레이어에게 **제시어의 글자 수**가 표시된다.

### 2. 그리기 + 맞추기

1. 출제자가 **설정된 시간** 동안 캔버스에 그림을 그린다.
2. 그리는 동안 다른 플레이어들은 **실시간으로 그림을 볼 수 있다**.
3. 나머지 플레이어들은 **채팅으로 정답을 입력**한다.
4. 정답을 맞추면 **시스템 메시지**로 순위와 점수가 표시되고, 이후 해당 플레이어는 채팅 입력이 비활성화된다.
5. 정답을 맞춘 순서대로 **1등 +3점, 2등 +2점, 3등 +1점**을 획득한다.
6. **3등이 나오면 즉시 라운드가 종료**된다. 시간이 끝나거나 모든 비출제자가 정답을 맞춰도 종료된다.
7. 비출제자가 3명 미만이면 있는 만큼만 점수가 부여되고, 전원 맞추면 종료된다.

### 3. 라운드 결과

1. 정답이 공개된다.
2. 점수가 부여된다 (아래 점수 참고).
3. **10초 카운트다운** 뒤 다음 라운드가 시작된다.

## 점수

| 상황 | 점수 |
|------|------|
| 1등 (가장 먼저 맞춘 플레이어) | +3점 |
| 2등 | +2점 |
| 3등 | +1점 |
| 4등 이후 | 없음 (3등 후 라운드 즉시 종료) |
| 못 맞춘 플레이어 | 0점 |
| 출제자 — 누군가 정답을 맞춘 경우 | +1점 |
| 출제자 — 아무도 못 맞춘 경우 | 0점 |

## 라운드 반복

설정된 라운드 수만큼 반복한다. 매 라운드마다 출제자가 순서대로 교체된다. 모든 라운드가 끝나면 **총점이 가장 높은 플레이어가 최종 승리**한다. 동점일 경우 공동 승리로 처리한다.

## 특이 사항

- 출제자는 채팅 입력이 비활성화된다 (정답 힌트 방지).
- 정답을 맞춘 플레이어의 채팅 메시지는 다른 플레이어에게 보이지 않는다 (정답 노출 방지).
- 출제자는 그림에 글자나 숫자를 쓸 수 없다 (그림으로만 표현).
