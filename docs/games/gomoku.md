# 오목 (Gomoku)

## 개요

2인 대전 보드 게임. 15×15 바둑판에서 번갈아 돌을 놓아 **5목을 먼저 완성**하면 승리한다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 2명 |
| 보드 크기 | 15×15 (총 225칸) |
| 돌 색상 | 흑(black), 백(white) |
| 선공 | 흑 |

## 게임 진행

1. 흑이 먼저 빈 칸에 돌을 놓는다.
2. 이후 흑과 백이 번갈아가며 빈 칸에 돌을 놓는다.
3. 이미 돌이 놓인 칸에는 둘 수 없다.
4. 자신의 차례가 아닐 때는 둘 수 없다.

## 승리 조건

가로, 세로, 대각선 중 하나의 방향으로 **같은 색 돌 5개 이상**을 연속으로 놓으면 승리한다.

### 판정 방향

| 방향 | 설명 |
|------|------|
| 가로 (→) | 같은 행에서 좌우로 연속 |
| 세로 (↓) | 같은 열에서 상하로 연속 |
| 대각선 (↘) | 우하향 대각선으로 연속 |
| 역대각선 (↗) | 우상향 대각선으로 연속 |

> 5개 이상이면 승리로 인정된다 (장목 허용).

## 무승부

225칸이 모두 채워졌는데 어느 쪽도 5목을 완성하지 못한 경우 무승부로 처리된다.

## 특이 사항

- 금수 규칙(삼삼, 사사 금지 등)은 적용되지 않는다.
- 장목(6목 이상)도 승리로 인정된다.
