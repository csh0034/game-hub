# 오목 (Gomoku)

## 개요

2인 대전 보드 게임. 15×15 바둑판에서 번갈아 돌을 놓아 **5목을 먼저 완성**하면 승리한다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 2명 |
| 보드 크기 | 15×15 (총 225칸) |
| 돌 색상 | 흑(black), 백(white) |
| 선공 | 흑 |
| 흑/백 배정 | 방장이 흑(기본) 또는 상대가 흑 선택 |
| 턴 제한 시간 | 10~60초 (기본 30초, 방 생성 시 설정) |
| 게임 규칙 | 자유룰(기본) 또는 렌주룰 선택 |

## 게임 진행

1. 흑이 먼저 빈 칸에 돌을 놓는다.
2. 이후 흑과 백이 번갈아가며 빈 칸에 돌을 놓는다.
3. 이미 돌이 놓인 칸에는 둘 수 없다.
4. 자신의 차례가 아닐 때는 둘 수 없다.
5. 각 턴은 설정된 제한 시간 이내에 두어야 한다. 시간이 초과되면 상대에게 턴이 넘어간다.

## 승리 조건

가로, 세로, 대각선 중 하나의 방향으로 **같은 색 돌 5개**를 연속으로 놓으면 승리한다.

### 판정 방향

| 방향 | 설명 |
|------|------|
| 가로 (→) | 같은 행에서 좌우로 연속 |
| 세로 (↓) | 같은 열에서 상하로 연속 |
| 대각선 (↘) | 우하향 대각선으로 연속 |
| 역대각선 (↗) | 우상향 대각선으로 연속 |

## 무승부

225칸이 모두 채워졌는데 어느 쪽도 5목을 완성하지 못한 경우 무승부로 처리된다.

## 게임 규칙

### 자유룰 (기본)

- 금수 규칙이 적용되지 않는다
- 삼삼, 사사 모두 허용된다
- 장목(6목 이상)도 승리로 인정된다

### 렌주룰

흑에게만 금수 규칙이 적용된다. 백은 제한 없이 자유롭게 둘 수 있다.

#### 금수 종류

| 금수 | 설명 |
|------|------|
| 장목 (6목 이상) | 한 방향으로 6개 이상 연속되면 금수 |
| 삼삼 (3-3) | 한 수로 활삼 2개 이상을 동시에 만들면 금수 |
| 사사 (4-4) | 한 수로 사 2개 이상을 동시에 만들면 금수 |

#### 금수 예외

해당 수를 놓았을 때 **정확히 5목**이 완성되면 금수가 아니다. 5목 완성은 모든 금수 규칙보다 우선한다.

#### 금수 표시

흑 차례에 금수 위치는 보드 위에 ✕ 표시로 확인할 수 있다. 금수 위치에는 돌을 놓을 수 없다.

| 대상 | 금수 표시 |
|------|----------|
| 흑 플레이어 | 표시됨 |
| 백 플레이어 | 표시 안 됨 |
| 관전자 | 표시됨 |

#### 백의 규칙

백에게는 금수 규칙이 적용되지 않는다. 장목(6목 이상)도 승리로 인정된다.

## 특이 사항

- 시간 초과 시 패배가 아닌 상대에게 턴이 넘어간다
