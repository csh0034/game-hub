# 라이어 드로잉 (Liar Drawing)

## 개요

3~8인 소셜 디덕션 게임. 제시어를 아는 시민들과 모르는 라이어가 섞여 **순서대로 그림을 그리고**, 투표로 라이어를 찾아낸다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 3~8명 |
| 라이어 수 | 1명 |
| 그리기 시간 | 1~120초 (방장 설정, 기본 60초) |
| 라운드 수 | 1~10회 (방장 설정, 기본 3라운드) |

## 게임 흐름

### 1. 라운드 시작 — 역할 배정

1. 플레이어 중 **1명이 라이어**로 랜덤 선정된다.
2. 모든 플레이어에게 **주제**(카테고리)가 공개된다.
3. **시민**에게는 제시어가 공개된다.
4. **라이어**에게는 제시어 대신 "당신은 라이어입니다"가 표시된다.

### 2. 그리기 턴

1. 플레이어 순서를 랜덤으로 결정한다.
2. 각 플레이어가 순서대로 **설정된 시간** 동안 캔버스에 그림을 그린다.
3. 그리는 동안 다른 플레이어들은 **실시간으로 그림을 볼 수 있다**.
4. 시간이 끝나면 자동으로 다음 플레이어로 넘어간다.
5. **"그리기 완료" 버튼**을 눌러 자신의 턴을 일찍 종료할 수 있다.
   - **완료**: 그림을 유지하고 턴을 넘긴다.
   - **스킵**: 그림을 지우고 턴을 넘긴다.
6. 모든 플레이어가 그린 후 그리기 턴이 종료된다.

### 3. 투표

1. 모든 플레이어의 그림을 한눈에 볼 수 있다.
2. 각 플레이어가 **라이어로 의심되는 1명**에게 투표한다 (자기 자신 제외).
3. 모든 플레이어가 투표하면 결과가 공개된다.
4. **최다 득표자가 라이어로 지목**된다.
5. 동률일 경우 **라이어 지목 실패**로 처리한다.

### 4. 라이어의 제시어 맞추기

- 라이어가 **지목된 경우에만** 진행된다.
- 라이어에게 **제시어를 맞출 기회**가 1회 주어진다.
- 제시어 제출 전 **확인 다이얼로그**가 표시된다.
- 정답을 맞추면 라이어가 역전 승리한다.

### 5. 결과 및 점수

라운드 결과 시 **정답(제시어)이 공개**되고 점수를 부여한다.

| 상황 | 시민 점수 | 라이어 점수 |
|------|-----------|-------------|
| 라이어 지목 성공 + 제시어 오답 | 각 +1점 | 0점 |
| 라이어 지목 성공 + 제시어 정답 | 0점 | +3점 |
| 라이어 지목 실패 | 0점 | +2점 |

#### 라운드 결과 문구

역할에 따라 다른 결과 문구가 표시된다.

| 상황 | 시민 | 라이어 | 관전자 |
|------|------|--------|--------|
| 라이어 적발 + 제시어 오답 | 라이어 적발! | 적발당했다! | 라이어 적발 |
| 라이어 적발 + 제시어 정답 | 라이어에게 역전당했다! | 역전 성공! | 라이어 역전! |
| 라이어 미적발 | 라이어를 놓쳤다! | 생존 성공! | 라이어 생존 |

## 라운드 반복

설정된 라운드 수만큼 반복한다. 매 라운드마다 라이어가 새로 선정된다. 라운드 결과 표시 후 **10초 카운트다운** 뒤 다음 라운드가 시작된다. 모든 라운드가 끝나면 **총점이 가장 높은 플레이어가 최종 승리**한다.

## 특이 사항

- 그리기 중 채팅은 비활성화된다.
- 라이어는 제시어를 모르지만 주제는 알므로, 주제에 맞는 그림으로 위장해야 한다.
- 시민은 제시어를 너무 명확하게 그리면 라이어에게 힌트를 줄 수 있으므로 적절히 애매하게 그려야 한다.
