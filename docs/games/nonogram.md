# 노노그램 (Nonogram)

## 개요

1인 퍼즐 게임. 격자의 행과 열에 주어진 숫자 힌트를 보고 **칸을 채워 숨겨진 그림을 완성**하면 승리한다. 5단계 난이도를 제공한다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 1명 |

## 난이도

| 난이도 | 격자 크기 | 총 칸 수 |
|--------|-----------|----------|
| 입문 (Tiny) | 5×5 | 25 |
| 초급 (Beginner) | 10×10 | 100 |
| 중급 (Intermediate) | 15×15 | 225 |
| 고급 (Expert) | 20×20 | 400 |
| 극한 (Extreme) | 40×40 | 1,600 |

## 규칙

### 힌트

- 각 행과 열에 숫자 힌트가 주어진다
- 숫자는 해당 행/열에서 **연속으로 채워진 칸의 그룹 크기**를 나타낸다
- 숫자가 여러 개이면 왼쪽(행) 또는 위쪽(열)부터 순서대로 나타나며, 각 그룹 사이에는 **최소 1칸의 빈 칸**이 있다
- 힌트가 `0`이면 해당 행/열에 채워진 칸이 없다

### 예시

힌트 `3 1`은 "연속 3칸이 채워진 후, 최소 1칸의 빈 칸을 두고, 1칸이 채워진다"를 의미한다.

## 게임 진행

1. 서버가 랜덤으로 퍼즐을 생성하고, 행/열 힌트를 제공한다
2. 플레이어는 힌트를 분석하여 칸을 채우거나 X 표시로 빈 칸을 확정한다
3. 실수에 대한 페널티는 없으며, 자유롭게 채우기/지우기가 가능하다
4. 채운 칸의 패턴이 정답과 일치하면 퍼즐이 완성된다

## 조작

| 동작 | 설명 |
|------|------|
| 좌클릭 | 칸 채우기 / 채운 칸 비우기 |
| 우클릭 | X 마킹 / X 마킹 해제 |
| 힌트 숫자 클릭 | 완료 체크 토글 (취소선 표시) |

- X 마킹은 빈 칸임을 확정하는 표시로, 풀이 보조 용도이다
- X 마킹된 칸은 좌클릭으로 채울 수 없으며, 먼저 X 마킹을 해제해야 한다
- 힌트 숫자를 클릭하면 취소선이 표시되어 이미 처리한 힌트를 구분할 수 있다. 풀이 보조 용도이며 승리 판정에 영향을 주지 않는다

## 승리 조건

채운 칸의 패턴이 정답과 정확히 일치하면 승리한다. X 마킹은 승리 판정에 영향을 주지 않는다.

## 화면 구성

- 상단에 진행률과 경과 시간을 표시한다
- 격자 상단에 열 힌트, 좌측에 행 힌트를 배치한다
