# 테트리스 (Tetris)

## 개요

1~8인 퍼즐 게임. 떨어지는 테트로미노를 회전·이동하여 **가로줄을 빈틈 없이 채우면** 해당 줄이 제거된다. 블록이 쌓여 보드 상단을 넘으면 게임 오버.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 1~8명 |
| 보드 크기 | 10×20 (가로×세로) |
| 난이도 | 초급 / 중급 / 고급 (기본 초급) |

## 테트로미노

7종류의 테트로미노가 존재하며, 각각 고유한 색상을 갖는다.

| 이름 | 모양 | 색상 |
|------|------|------|
| I | `████` (1×4) | 하늘 |
| O | `██` / `██` (2×2) | 노랑 |
| T | `███` / `_█_` | 보라 |
| S | `_██` / `██_` | 초록 |
| Z | `██_` / `_██` | 빨강 |
| J | `█__` / `███` | 파랑 |
| L | `__█` / `███` | 주황 |

피스는 **7-bag 랜덤** 방식으로 생성된다. 7종류를 한 세트로 섞어 순서대로 제공하며, 한 세트가 소진되면 새로운 세트를 생성한다.

## 게임 진행

1. 테트로미노가 보드 상단 중앙에 생성된다.
2. 피스는 일정 간격으로 한 칸씩 아래로 떨어진다.
3. 플레이어는 피스를 좌우 이동, 회전, 소프트 드롭, 하드 드롭할 수 있다.
4. 피스가 바닥이나 다른 블록 위에 착지하면 **1틱의 유예 시간**(Lock Delay) 후 고정(lock)된다. 유예 중 이동/회전하면 타이머가 리셋된다.
5. 가로줄이 빈틈 없이 채워지면 해당 줄이 제거되고 위의 블록이 내려온다.
6. 다음 피스가 생성되며 반복한다.

## 조작

| 동작 | 설명 |
|------|------|
| ← / → | 좌우 이동 |
| ↑ 또는 X | 시계 방향 회전 |
| Z | 반시계 방향 회전 |
| ↓ | 소프트 드롭 (빠르게 내리기) |
| Space | 하드 드롭 (즉시 착지) |
| C 또는 Shift | 홀드 |

## 점수 시스템

### 기본 줄 클리어

줄 클리어 시 기본 점수에 레벨 배율을 곱한다.

| 줄 수 | 이름 | 기본 점수 |
|-------|------|-----------|
| 1줄 | Single | 100 |
| 2줄 | Double | 300 |
| 3줄 | Triple | 500 |
| 4줄 | Tetris | 800 |

> **점수 = 기본 점수 × 레벨**

- 소프트 드롭: 낙하한 칸 수만큼 1점씩 추가
- 하드 드롭: 낙하한 칸 수만큼 2점씩 추가

### T-Spin 보너스

T피스를 **회전**으로 끼워넣으면 T-Spin으로 판정된다. 판정 조건: T피스 중심의 대각선 4코너 중 **3개 이상이 점유**(블록 또는 벽)되어 있어야 한다.

| 줄 수 | 이름 | 기본 점수 | 쓰레기 줄 (대전) |
|-------|------|-----------|------------------|
| 0줄 | T-Spin Mini | 0 | 0줄 |
| 1줄 | T-Spin Single | 800 | 2줄 |
| 2줄 | T-Spin Double | 1200 | 4줄 |
| 3줄 | T-Spin Triple | 1600 | 6줄 |

### 콤보 (Combo)

연속으로 줄을 클리어하면 콤보 카운트가 1씩 증가한다. 줄을 클리어하지 못하면 콤보가 0으로 리셋된다.

- **콤보 보너스 점수**: 50 × 콤보 수 × 레벨
- **대전 시 추가 쓰레기**: 콤보 4 이상부터 (콤보 수 - 3)줄 추가 전송

### Back-to-Back (B2B)

**어려운 클리어**(Tetris 또는 T-Spin 클리어)를 연속으로 달성하면 Back-to-Back 보너스가 적용된다. 일반 클리어(1~3줄, T-Spin 아님)가 중간에 발생하면 체인이 끊어진다.

- **점수 보너스**: 1.5배 (두 번째 연속 어려운 클리어부터)
- **대전 시 추가 쓰레기**: +1줄

## 레벨과 속도

### 난이도별 초기 설정

| 난이도 | 초기 레벨 | 초기 낙하 간격 |
|--------|-----------|----------------|
| 초급 | 1 | 0.8초 |
| 중급 | 1 | 0.6초 |
| 고급 | 5 | 0.4초 |

- 10줄을 클리어할 때마다 레벨이 1 상승한다.
- 레벨이 오를수록 낙하 간격이 짧아진다.

## 게임 오버 조건

새로운 피스가 생성될 때 기존 블록과 겹치면 게임 오버된다 (**탑아웃**).

## 게임 모드

### 솔로 모드

- 1인 플레이. 최고 점수를 목표로 한다.
- 게임 오버 시 최종 점수가 기록된다.

### 대전 모드

- 2~8인 대전. 각자의 보드에서 동시에 플레이한다.
- **화면 구성**: 내 보드가 왼쪽에 크게, 상대 보드가 오른쪽에 2열 그리드로 배치된다.
- 상대 수가 많을수록 상대 보드가 작아진다 (1명 16px, 2명 14px, 3명 12px, 4~5명 10px, 6~7명 8px).
- **공격 줄**: 줄 클리어 시 아래 표에 따라 다른 모든 상대 보드 하단에 쓰레기 줄이 추가된다.
- T-Spin, 콤보, Back-to-Back 보너스로 추가 쓰레기 줄이 전송될 수 있다.
- 쓰레기 줄은 한 칸의 빈 공간을 제외하고 모두 채워진 상태로 추가된다.
- 마지막까지 살아남은 플레이어가 승리한다.

| 클리어 유형 | 기본 쓰레기 줄 | 비고 |
|------------|---------------|------|
| 1줄 (Single) | 0줄 | 공격 없음 |
| 2줄 (Double) | 1줄 | |
| 3줄 (Triple) | 2줄 | |
| 4줄 (Tetris) | 3줄 | B2B 시 +1줄 |
| T-Spin Single | 2줄 | B2B 시 +1줄 |
| T-Spin Double | 4줄 | B2B 시 +1줄 |
| T-Spin Triple | 6줄 | B2B 시 +1줄 |
| 콤보 4+ | +（콤보 수 - 3）줄 | 기본 공격에 추가 |

## 특수 규칙

### 홀드 (Hold)

- 현재 피스를 홀드 슬롯에 저장하고, 이전에 저장된 피스를 꺼낸다.
- 홀드 슬롯이 비어있으면 다음 피스가 생성된다.
- 한 턴에 한 번만 홀드할 수 있다. 피스를 고정한 후 다시 사용 가능해진다.

### 넥스트 미리보기 (Next Preview)

- 다음에 나올 피스를 **2개까지** 미리 볼 수 있다.

### 고스트 피스 (Ghost Piece)

- 현재 피스가 하드 드롭될 위치를 반투명하게 표시한다.
- 정확한 착지 위치를 예측하는 데 도움을 준다.
- **내 보드에서만** 표시된다. 상대 보드에는 고스트 피스가 표시되지 않는다.

### 벽차기 (Super Rotation System, SRS)

- 피스 회전 시 벽이나 다른 블록에 막히면, 미리 정의된 오프셋 위치로 이동을 시도한다.
- 최대 5가지 오프셋을 순서대로 시도하며, 하나라도 성공하면 회전이 적용된다.
- 모든 오프셋이 실패하면 회전이 취소된다.

### T-Spin

- T피스를 **회전**으로 끼워넣으면 T-Spin으로 판정된다.
- **3-corner 판정**: T피스 중심의 대각선 4코너 중 3개 이상이 블록 또는 벽으로 점유되어 있어야 한다.
- T-Spin으로 줄을 클리어하면 일반 클리어보다 높은 점수와 쓰레기 줄을 획득한다.
- 줄 클리어 없이 T-Spin만 발생하면 T-Spin Mini로 표시된다 (추가 점수 없음).

### 콤보 (Combo)

- 연속으로 줄을 클리어할 때마다 콤보 카운트가 1씩 증가한다.
- 줄을 클리어하지 못하고 피스가 고정되면 콤보가 0으로 리셋된다.
- 콤보 보너스: 50 × 콤보 수 × 레벨 (추가 점수).
- 대전 시: 콤보 4 이상부터 (콤보 수 - 3)줄의 추가 쓰레기 전송.

### Back-to-Back (B2B)

- **어려운 클리어**(Tetris 4줄 또는 T-Spin 클리어)를 연속으로 달성하면 보너스가 적용된다.
- 일반 클리어(1~3줄, T-Spin 아님)가 중간에 발생하면 체인이 끊어진다.
- 점수 1.5배, 대전 시 쓰레기 +1줄.

### Lock Delay (잠금 지연)

- 피스가 바닥이나 다른 블록 위에 도달하면 **1틱의 유예 시간**이 주어진다.
- 유예 시간 동안 좌우 이동/회전하면 타이머가 리셋되어 추가 조작이 가능하다.
- 리셋은 **최대 5회**까지만 허용된다. 5회 초과 시 다음 틱에 즉시 고정된다.
- 피스가 아래로 낙하에 성공하면 리셋 횟수가 초기화된다.
- 유예 시간이 지나도록 조작이 없으면 피스가 고정된다.
- 하드 드롭은 Lock Delay를 무시하고 즉시 고정한다.
