# 타자 게임 (Typing)

1~8인 타자 레이스. 위에서 내려오는 단어를 빠르게 입력하여 제거하는 게임.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 1~8명 |
| 난이도 | 초급 / 중급 / 고급 (기본 초급) |
| 제한 시간 | 1~120초 (기본 60초) |
| 목숨 | 1~10개 (기본 3개) |

## 게임 진행

1. 게임 시작 시 화면 상단에서 단어가 내려오기 시작한다
2. 모든 플레이어에게 동일한 단어가 동일한 타이밍에 내려온다
3. 하단 입력창에 단어를 타이핑하고 Enter로 제출한다
4. 정확히 일치하면 자신의 화면에서만 해당 단어가 제거된다
5. 시간이 지날수록 낙하 속도와 출현 빈도가 증가한다

## 단어

- 현재 한글 단어만 지원
- 난이도별 단어 글자 수가 달라짐
- 화면에 동시에 표시되는 단어 수는 난이도에 따라 제한됨
- 같은 단어가 화면에 중복 출현하지 않음

## 난이도

| 난이도 | 글자 수 | 초기 낙하 시간 | 최대 동시 단어 수 | 출현 간격 |
|--------|---------|---------------|------------------|----------|
| 초급 | 2~3자 | 6초 | 5개 | 2.0초 |
| 중급 | 2~4자 | 4.5초 | 7개 | 1.5초 |
| 고급 | 2~5자 | 3초 | 10개 | 1.0초 |

- 낙하 시간: 단어가 상단에서 하단까지 도달하는 데 걸리는 시간
- 시간 경과에 따라 낙하 속도와 출현 빈도가 점진적으로 빨라짐

| 구간 | 경과 비율 | 낙하 속도 배율 | 출현 간격 배율 |
|------|----------|---------------|---------------|
| 초반 | 0~33% | ×1.0 | ×1.0 |
| 중반 | 34~66% | ×1.25 | ×0.85 |
| 후반 | 67~100% | ×1.5 | ×0.7 |

## 목숨과 탈락

목숨이 0이 되면 탈락한다. 목숨이 줄어드는 조건은 두 가지:

| 조건 | 설명 | UI 피드백 |
|------|------|----------|
| 놓침 | 단어가 화면 하단에 도달 | 바닥 빨간 번쩍임 + "놓침!" 표시 |
| 오타 | Enter로 제출한 단어가 화면의 어떤 단어와도 일치하지 않음 | 입력창 빨간 흔들림 + "오타!" 표시 |

- 탈락한 플레이어의 화면에 "탈락" 표시
- 탈락 후에도 다른 플레이어의 진행 상황을 관전 가능

## 점수

단어를 정확히 입력하면 글자 수에 비례한 점수를 획득한다.

| 글자 수 | 점수 |
|---------|------|
| 2자 | 100 |
| 3자 | 200 |
| 4자 | 350 |
| 5자 | 500 |

- 연속으로 단어를 맞히면 콤보 카운트가 증가하고, 5콤보마다 보너스 점수 획득
  - 5콤보: +200, 10콤보: +500, 20콤보: +1000, 이후 10콤보마다: +1000
- 목숨을 잃으면 콤보가 초기화됨

## 게임 종료

게임은 아래 조건 중 하나가 충족되면 종료된다:

- 제한 시간이 모두 소진됨
- 모든 플레이어가 탈락함

## 승리 판정

- 최고 점수를 기록한 플레이어가 승리
- 동점이면 공동 승리
- 모든 플레이어가 탈락한 경우에도 점수가 가장 높은 플레이어가 승리

## 화면 구성

### 플레이어 화면

- 내 게임 영역이 왼쪽에 크게 표시 (단어 낙하 영역 + 하단 입력창)
- 상대 플레이어 현황이 오른쪽에 요약 표시 (닉네임, 점수, 남은 목숨, 상태)
- 상단에 남은 시간, 내 점수, 남은 목숨, 콤보 표시

### 관전자 화면

- 모든 플레이어의 점수, 목숨, 상태를 한눈에 볼 수 있는 전체 현황판
- 1등 플레이어의 게임 화면을 메인으로 표시

## 관전

- 비공개 정보 없음 (안내 배너 미표시)
