# 용어 사전

## 공통 

| 한글    | 영어 | 설명 |
|-------|------|------|
| 게임허브  | Game Hub | 웹 기반 멀티플레이 게임 플랫폼. 로비에서 방을 만들고 다양한 게임을 플레이할 수 있다 |
| 로비    | Lobby | 방 목록을 확인하고 방을 생성하거나 입장할 수 있는 공간 |
| 방     | Room | 플레이어들이 모여 게임을 진행하는 단위 |
| ↳ 대기실 | Waiting Room | 방 입장 후 게임 시작 전 플레이어들이 준비하는 대기 화면 |
| ↳ 게임룸 | Game Room | 게임이 진행되는 화면 |
| 로비 채팅 | Lobby Chat | 로비에서 인증된 모든 유저가 참여하는 채팅. 서버가 최근 50개 메시지 이력을 보관하며, 입장 시 이력을 전송 |
| 방 채팅  | Room Chat | 같은 방에 있는 플레이어 간 채팅. 대기실/게임 중 모두 사용 가능. 서버가 최근 50개 메시지 이력을 보관하며, 입장 시 이력을 전송. 방 삭제 시 이력도 삭제 |
| 귓속말 | Whisper | 특정 사용자에게 1:1로 보내는 비밀 메시지. 채팅창에 `@닉네임 메시지` 형태로 전송하며, 이력을 저장하지 않는 휘발성 메시지 |

## 게임

| 한글 | 영어 | 설명 |
|------|------|------|
| 노노그램 | Nonogram | 1인 퍼즐. 행/열 힌트를 보고 칸을 채워 그림을 완성하면 승리 |
| 타자 게임 | Typing | 1~8인 타자 레이스. 위에서 내려오는 단어를 입력하여 제거하는 게임 |
| 라이어 드로잉 | Liar Drawing | 3~8인 소셜 디덕션 그림 게임. 제시어를 모르는 라이어를 투표로 찾아내기 |
| 캐치마인드 | Catch Mind | 3~8인 드로잉 퀴즈. 출제자가 그린 그림을 보고 채팅으로 정답을 맞추는 게임 |
| 테트리스 | Tetris | 1~8인 대전. 10×20 보드에서 블록을 쌓고, 대전 시 쓰레기 줄 공격 |
| 오목 | Gomoku | 2인 대전. 15×15 보드에서 5목을 먼저 완성하면 승리 |
| 지뢰찾기 | Minesweeper | 1인 퍼즐. 지뢰를 피해 모든 안전한 칸을 열면 승리 |

## 방 상태

| 한글 | 영어 | 설명 |
|------|------|------|
| 대기 중 | waiting | 방이 생성된 후 게임 시작 전 플레이어를 모집하는 상태 |
| 게임 중 | playing | 게임이 진행되고 있는 상태 |
| 게임 종료 | finished | 게임이 끝난 상태. 방장이 "다시하기"를 누르면 waiting으로 복귀 |

## 난이도

| 한글 | 영어 | 설명 |
|------|------|------|
