# 강퇴 (Kick)

방장이 대기실에서 플레이어 또는 관전자를 내보낼 수 있다.

## 핵심 규칙

| 항목 | 내용 |
|------|------|
| 권한 | 방장만 가능 |
| 가능 상태 | waiting 상태에서만 (playing/finished 불가) |
| 대상 | 플레이어, 관전자 모두 |
| UI | 플레이어/관전자 목록 우측 X 버튼 (방장에게만 노출) |
| 확인 | 확인 다이얼로그 필수 ("닉네임 님을 내보내시겠습니까?") |
| 강퇴 후 | 대상은 로비로 자동 이동, 토스트 메시지 표시 |

## 관전자 일괄 강퇴

- 관전 허용 ON → OFF 전환 시 관전자가 있으면 "관전자를 내보내시겠습니까?" 확인 다이얼로그 표시
  - 확인: 관전자 전원 강퇴 + OFF 전환
  - 취소: ON 유지
