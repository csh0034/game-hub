# 닉네임

계정 시스템 없이 닉네임만으로 인증하는 간편 로그인. 닉네임을 입력하면 즉시 로비에 입장한다.

## 닉네임 규칙

| 항목 | 내용 |
|------|------|
| 글자 수 | 3~20자 (앞뒤 공백 자동 제거) |
| 예약어 | "관리자" 사용 불가 |
| 고유성 | 동시에 같은 닉네임 사용 불가 |

## 로그인 흐름

1. 닉네임 입력 후 확인을 누르면 서버가 유효성을 검사한다.
2. 사용 가능한 닉네임이면 즉시 인증되어 로비에 입장한다.
3. 닉네임은 브라우저에 저장되어, 페이지 새로고침 시 자동으로 재인증한다.

## 닉네임 충돌 처리

같은 닉네임이 이미 사용 중일 때, 브라우저를 기준으로 처리가 달라진다.

- 같은 브라우저에서 접속한 경우 (새 탭, 새로고침 등): 기존 연결을 종료하고 새 연결로 교체한다. 방에 있었다면 해당 방에 자동 복귀한다.
- 다른 브라우저에서 접속한 경우: "이미 사용 중인 닉네임입니다" 안내와 함께 거부된다.

## 강제 로그아웃

같은 브라우저의 다른 탭에서 동일 닉네임으로 접속하면, 기존 탭에 "다른 탭에서 접속하여 로그아웃되었습니다" 알림이 표시되고 로그인 화면으로 돌아간다.

## 재접속

- 페이지 새로고침 시 저장된 닉네임으로 자동 재인증한다.
- 방에 있던 중 재접속하면 해당 방에 자동 복귀한다.

## 로그아웃

로그아웃 버튼을 누르면 닉네임이 해제되고 로그인 화면으로 돌아간다.

## 관리자

지정된 닉네임으로 로그인하면 관리자 권한이 부여되어 일반 사용자에게 노출되지 않는 관리 기능을 사용할 수 있다. "관리자"라는 닉네임 자체는 예약어로 사용할 수 없다.

- 관리자의 실제 닉네임은 모든 화면에서 "관리자"로 표시된다
  - 채팅, 방 목록, 대기실, 게임 내, 온라인 플레이어 목록, 랭킹 등
- 관리자 닉네임은 강조색으로 구분된다
