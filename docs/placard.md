# 플랜카드

관리자가 로비 상단에 짧은 안내 문구를 표시하는 배너 기능.

## 규칙

| 항목 | 내용 |
|------|------|
| 위치 | 로비 탭 네비게이션과 빠른 시작 타이틀 사이 |
| 접근 | 관리자 전용 메뉴 "플랜카드" (공지하기 뒤에 배치) |
| UI | 메뉴 클릭 시 다이얼로그 (문구 목록 관리, 항목당 50자 제한) |
| 등록 수 | 최대 5개 |
| 노출 | 등록된 문구가 있을 때만 모든 사용자에게 배너 표시 |
| 비우기 | 모든 문구를 삭제하면 배너 영역이 사라짐 |
| 저장 | 서버에 저장 (새로고침·재접속 시 유지) |

## 슬라이드

- 1개 등록 시 정적 배너로 표시
- 2개 이상 등록 시 4초 간격으로 자동 슬라이드
- 하단 도트 인디케이터로 현재 위치 표시
- 마우스를 올리면 슬라이드가 일시정지

## 용도

- 상시 안내 배너 (예: 점검 예정, 새 게임 추가)
- 이벤트·축하 알림 (예: 랭킹 달성 축하)
- 여러 안내를 동시에 순환 표시
