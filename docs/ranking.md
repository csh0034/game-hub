# 랭킹 시스템

1인 플레이 게임(지뢰찾기, 테트리스 솔로)의 난이도별 Top 10 랭킹.

## 대상 게임

| 게임 | 기준 | 정렬 |
|------|------|------|
| 지뢰찾기 | 완료 시간 | 낮을수록 좋음 |
| 테트리스 클래식 솔로 | 점수 | 높을수록 좋음 |
| 테트리스 스피드 레이스 솔로 | 클리어 시간 | 낮을수록 좋음 |

- 테트리스 클래식은 솔로 모드에서 게임오버 시 점수 기준으로 랭킹 등록
- 테트리스 스피드 레이스는 솔로 모드에서 40줄 클리어 시에만 랭킹 등록
- 대전 모드는 제외

## 난이도별 분리

지뢰찾기 3종 + 테트리스 클래식 3종 + 테트리스 스피드 3종 = 총 9개 랭킹

## 기본 규칙

- 난이도당 최대 10개 엔트리 저장
- 식별자는 닉네임 (계정 시스템 없음)
- 같은 닉네임은 난이도당 최고 기록 1개만 유지, 더 좋은 기록이면 갱신
- 게임 종료 시 서버가 자동으로 랭킹 등록 (치팅 방지)
- 랭킹 변경 시 로비 및 대기실의 랭킹 카드가 실시간 갱신

## 치팅 방지

### 공통

- 같은 닉네임의 중복 등록 차단, 기존보다 나은 기록만 갱신

### 지뢰찾기

- 최소 클리어 시간 미만은 랭킹 등록 거부: 초급 3초, 중급 15초, 고급 35초
- 칸 열기/양클릭 간 최소 50ms 간격 강제
- 난이도별 최소 클릭 수 미달 시 거부: 초급 3회, 중급 10회, 고급 15회
- 클릭 간격이 비정상적으로 균일하면 거부 (봇 탐지)
- 대부분의 클릭이 최소 간격 근처이면 거부
- 난이도별 사고 구간(긴 멈춤) 부족 시 거부: 중급 1회, 고급 2회 이상

### 테트리스 클래식

- 최소 점수 미만은 랭킹 등록 거부: 전 난이도 100점

### 테트리스 스피드 레이스

- 최소 클리어 시간 미만은 랭킹 등록 거부: 초급 20초, 중급 15초, 고급 10초

## 관리자 기능

- 관리자가 로비 랭킹 카드에서 의심 기록을 삭제 가능

## UI 위치

- 로비: 채팅 패널 하단 랭킹 카드
- 대기실: 게임 옵션 아래 랭킹 카드
- 게임 결과: 순위 표시, 1위 시 신기록 표시
