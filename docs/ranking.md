# 랭킹 시스템

1인 플레이 게임(지뢰찾기, 테트리스 솔로)의 난이도별 Top 10 랭킹.

## 대상 게임

| 게임 | 기준 | 정렬 |
|------|------|------|
| 지뢰찾기 | 완료 시간 | 낮을수록 좋음 |
| 테트리스 스피드 레이스 솔로 | 클리어 시간 | 낮을수록 좋음 |

- 테트리스는 스피드 레이스 솔로 모드에서 40줄 클리어 시에만 랭킹 등록
- 클래식 모드 및 대전 모드는 제외

## 난이도별 분리

지뢰찾기 3종 + 테트리스 3종 (beginner/intermediate/expert) = 총 6개 랭킹

## 기본 규칙

- 난이도당 최대 10개 엔트리 저장
- 식별자는 닉네임 (계정 시스템 없음)
- 게임 종료 시 서버가 자동으로 랭킹 등록 (치팅 방지)
- 랭킹 변경 시 로비 및 대기실의 랭킹 카드가 실시간 갱신

## 치팅 방지

- 지뢰찾기 최소 클리어 시간 미만은 랭킹 등록 거부: 초급 3초, 중급 15초, 고급 35초
- 칸 열기 간 최소 50ms 간격 강제 (자동화 클릭 방지)

## 관리자 기능

- 관리자가 로비 랭킹 카드에서 의심 기록을 삭제 가능

## UI 위치

- 로비: 채팅 패널 하단 랭킹 카드
- 대기실: 게임 옵션 아래 랭킹 카드
- 게임 결과: 순위 표시, 1위 시 신기록 표시
