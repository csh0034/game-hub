# 방 운영 규칙

## 중도 이탈 처리

게임 진행 중 플레이어가 나가면 해당 게임은 즉시 종료되고, 남은 모든 플레이어는 5초 후 로비로 이동한다. 인원 수나 게임 종류에 관계없이 동일하게 적용된다.

## 다시하기

게임 시작 후 방장에게 "다시하기" 버튼이 항상 노출된다. 게임 종료 후에는 즉시 실행되고, 게임 진행 중에는 확인 다이얼로그를 거친다. 방장이 누르면 방의 모든 플레이어와 관전자가 대기실로 복귀한다.

## 방 이름 수정

방장이 대기실에서 방 이름을 변경할 수 있다. 1~20자, 대기실과 로비 방 목록에 실시간 반영된다.
