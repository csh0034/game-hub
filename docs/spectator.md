# 관전 모드 (Spectator Mode)

모든 게임에 적용되는 관전 기능. 관전자는 게임 진행을 지켜보며 채팅에 참여할 수 있다.

## 핵심 규칙

| 항목 | 내용 |
|------|------|
| 대상 게임 | 모든 게임 |
| 최대 관전자 | 4명 |
| 관전 ON/OFF | 호스트가 게임 옵션에서 설정, 기본값 OFF |
| 관전자 채팅 ON/OFF | 호스트가 게임 옵션에서 설정, 기본값 OFF (관전 ON일 때만 노출) |
| 입장 시점 | waiting 상태에서만 관전 입장 가능 |
| 게임 종료 후 | 관전자는 웨이팅룸에 잔류 |
| 관전자 이탈 | 게임 종료 트리거하지 않음 |
| 비공개 정보 | 관전자에게 모두 공개 (안내 문구 표시) |

## 관전 허용 옵션

- 호스트가 대기실에서 "관전 허용" ON/OFF 토글로 설정
- UI 노출 순서: ON → OFF
- 기본값: OFF
- 관전 ON → OFF 전환 시 관전자가 있으면 "관전자를 내보내시겠습니까?" 확인 다이얼로그 표시
  - 확인: 관전자 전원 퇴장 + OFF 전환
  - 취소: ON 유지

## 관전자 채팅

- 호스트가 "관전자 채팅" ON/OFF 토글로 설정 (관전 허용 ON일 때만 노출)
- 기본값: OFF
- OFF이면 관전자 채팅 메시지를 서버에서 차단, 클라이언트 입력창 비활성화
- ON이면 관전자 메시지에 `[관전]` 표시 추가

## 비공개 정보 공개

관전자에게 "관전자 모드: 모든 정보가 공개됩니다" 안내 배너를 표시하고, 아래 정보를 공개한다:

| 게임 | 공개 정보 | 안내 배너 |
|------|----------|----------|
| 오목 | 해당 없음 (공개 게임) | 미표시 |
| 지뢰찾기 | 해당 없음 (1인 게임) | 미표시 |
| 테트리스 | 해당 없음 | 미표시 |
| 텍사스 홀덤 | 모든 플레이어의 홀카드 | 표시 |
| 라이어 드로잉 | 라이어 정체 + 제시어 | 표시 |
| 캐치마인드 | 제시어 | 표시 |

비공개 정보가 있는 게임(홀덤, 라이어드로잉, 캐치마인드)에서만 "관전자 모드: 모든 정보가 공개됩니다" 안내 배너가 표시된다.

## URL 직접 접속

- `/room/[id]` URL로 접속 시 playing 상태면 로비로 리다이렉트
- 관전 OFF면 일반 참가만 가능 (관전 입장 불가)
