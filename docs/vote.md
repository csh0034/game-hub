# 투표

관리자가 선택지를 등록하면 로비 접속자들이 복수 투표하는 기능.

## 현재 용도

디자인 컨셉 투표. 6종의 사이트 디자인 목업에 대해 선호도를 수집한다.

## 규칙

| 항목 | 내용 |
|------|------|
| 위치 | 로비 우측 사이드바 (채팅 패널 아래) |
| 투표 방식 | 복수 투표 (여러 항목에 좋아요 토글) |
| 투표 식별 | 브라우저 ID 기반 (닉네임 변경에 영향 없음) |
| 미리보기 | 항목 클릭 시 새 탭에서 HTML 파일 열기 |
| 저장 | 서버에 저장 (새로고침·재접속 시 유지) |
| 실시간 | 투표 시 모든 접속자에게 즉시 반영 |

## 표시

- 각 항목마다 하트 토글 버튼, 투표 수, 항목 이름, 미리보기 링크를 한 줄로 표시
- 본인이 투표한 항목은 하트가 채워진 상태로 표시
- 미리보기 아이콘 클릭 시 서버에서 제공하는 HTML 파일이 새 탭에 열림

## 향후 계획

- 관리자가 투표 주제와 선택지를 자유롭게 등록/관리하는 범용 투표 기능으로 확장
- 투표 생성, 마감, 결과 조회 등 관리자 기능 추가
